Important Concepts

World Wide Internet

The web is a bunch of computers that communicate with each other. When a person goes to a web page like www.google.com, their computer sends a HTTP Request to a server. The server finds the appropriate HTML document and sends it back to the user's computer where a web browser interprets the page and displays it on the user's screen.

1. You utilize a brower which runs a program on your computer to display files that make up the web.
2. HTTP is the main protocol for the web.
3. Servers are computers that host files that make up the web.
4. Internet is the world's largest computer network.

HTML - The heart of the web!

HTML stands for Hypertext Markup Language. HTML documents form the majority of the content on the web and is made up of the following:

1. Text Content (what you see)
2. Mark up (what it looks like)
3. Reference to other documents (such as images and videos)
4. Links to other pages

Why Computers are Stupid

Computers are stupid because they interpret instructions literally. This makes them very unforgiving since a small mistake by a programmer can cause huge problems in a program so it is very important to verify code while writing.

Inline vs Block Elements

HTML elements are either inline or block. Block elements form an "invisible box" around the content inside of them. Inline tags include anchor, break, image, and strong while block tags include paragraph, form, and div.
